# Vue Training

## Attribute Binding

- v-bind dynamically binds and attribute to an expression
- v-bind:src="expression"

```javascript
<img v-bind:src="image" />
```

#### shorthand

- :alt="desc"
- :href="url"
- :title="toolTip"
- :class="isActive"
- :style="isStyled"
- :disabled="isDisabled"

```javascript
<img :src="image" />
```
